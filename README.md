# dados-ibge
Aplicação que importa dados da API do IBGE (https://servicodados.ibge.gov.br/api/docs/localidades) e os armazena em um banco de dados NoSQL (MongoDB).


- Regiões
- Estados
- Cidades


# Ferramentas/ Tecnologias utilizadas:

- Java 17 (Spring boot)
- Rest API
- Feign Client
- MongoDB
- RabbitMQ
- Docker
- JUnit
- Mockito

# Executando localmente

- mvn clean package
- docker-compose up -d
